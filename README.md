# RestroDiningHub

A Restaurant Management and Online Food Ordering System developed using Django.

## Features

* User Registration and Login
* Restaurant Listing
* Restaurant Details
* Food Menu Management
* Cart Management
* Wishlist Management
* Table Booking
* Order Management
* Payment Module

## Technologies Used

* Python
* Django
* HTML
* CSS
* JavaScript
* SQLite

## Installation

```bash
git clone https://github.com/zalaba1610/RestroDiningHub.git
cd RestroDiningHub
python manage.py runserver
```
